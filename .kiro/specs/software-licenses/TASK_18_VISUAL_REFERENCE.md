## User Experience Flow

### Loading Flow
1. User navigates to page
2. Next.js shows loading.tsx immediately
3. Skeleton components render
4. Data fetches in background
5. Page transitions to actual content
6. Smooth, no layout shift

### Error Flow
1. User navigates to page
2. Error occurs during data fetch
3. Error boundary catches error
4. Error page displays with clear message
5. User can retry or navigate away
6. Error logged for monitoring

### Not Found Flow
1. User navigates to invalid license ID
2. Loader calls `notFound()`
3. Not found page displays
4. User can navigate back to list
5. Clear messaging about issue
