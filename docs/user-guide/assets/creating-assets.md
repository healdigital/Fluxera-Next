# Creating Assets

Learn how to add new assets to your inventory and manage asset information effectively.

## Overview

Creating assets in Fluxera is straightforward. You can add detailed information about each asset including specifications, purchase details, and warranty information.

## Step-by-Step Guide

### 1. Navigate to Assets

From the main navigation menu, click on **Assets** to open the asset management page.

### 2. Click Create Asset

Look for the **Create Asset** button in the top-right corner of the page and click it.

![Create Asset Button](../images/create-asset-button.png)

### 3. Fill in Asset Information

Complete the asset creation form with the following information:

#### Required Fields

- **Name**: A descriptive name for the asset (e.g., "MacBook Pro 16-inch")
- **Category**: Select the asset type from the dropdown
  - Laptop
  - Desktop
  - Monitor
  - Peripheral
  - Mobile Device
  - Networking
  - Other
- **Status**: Current status of the asset
  - Available
  - Assigned
  - In Maintenance
  - Retired

#### Optional Fields

- **Serial Number**: Manufacturer's serial number (recommended for warranty tracking)
- **Model**: Specific model number or name
- **Manufacturer**: Brand or manufacturer name
- **Purchase Date**: When the asset was acquired
- **Purchase Price**: Cost of the asset
- **Warranty Expiration**: When the warranty expires
- **Location**: Physical location of the asset
- **Notes**: Additional information or special instructions

### 4. Upload Asset Image (Optional)

You can upload a photo of the asset to make it easier to identify:

1. Click the **Upload Image** button
2. Select an image file (JPG, PNG, or WebP)
3. The image will be automatically optimized for performance

### 5. Save the Asset

Click the **Create Asset** button at the bottom of the form to save your new asset.

You'll see a success message confirming the asset was created, and you'll be redirected to the asset list where your new asset will appear.

## Viewing Asset Details

After creating an asset, you can view its full details:

1. Click on the asset name in the asset list
2. View all asset information, assignment history, and maintenance records
3. Use the tabs to navigate between different sections:
   - **Details**: Basic asset information
   - **History**: Complete audit trail of changes
   - **Maintenance**: Scheduled and completed maintenance

## Editing Assets

To modify an existing asset:

1. Navigate to the asset detail page
2. Click the **Edit** button
3. Update the information as needed
4. Click **Save Changes**

All changes are automatically logged in the asset history.

## Filtering and Searching

### Search Assets

Use the search bar at the top of the asset list to find assets by:
- Name
- Serial number
- Model
- Manufacturer

### Filter Assets

Use the filter options to narrow down the asset list:

- **Category**: Show only specific asset types
- **Status**: Filter by availability status
- **Location**: View assets in specific locations
- **Date Range**: Filter by purchase date

### Sort Assets

Click on column headers to sort assets by:
- Name (A-Z or Z-A)
- Purchase Date (newest or oldest)
- Status
- Category

## Bulk Operations

### Importing Assets

To import multiple assets at once:

1. Click the **Import** button
2. Download the CSV template
3. Fill in asset information in the template
4. Upload the completed CSV file
5. Review and confirm the import

### Exporting Assets

To export your asset inventory:

1. Apply any desired filters
2. Click the **Export** button
3. Choose your format (CSV or Excel)
4. The file will download automatically

## Common Questions

**Q: Can I create assets without serial numbers?**
A: Yes, serial numbers are optional, but highly recommended for tracking and warranty purposes.

**Q: What image formats are supported?**
A: JPG, PNG, and WebP formats are supported. Images are automatically optimized.

**Q: Can I edit an asset after creating it?**
A: Yes, you can edit any asset at any time. All changes are logged in the history.

**Q: How do I delete an asset?**
A: Open the asset detail page and click the **Delete** button. Note: This action cannot be undone.

**Q: Can I assign an asset during creation?**
A: No, assets must be created first, then assigned separately. This ensures proper tracking.
