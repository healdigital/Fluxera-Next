# User Management

The User Management module allows you to invite team members, assign roles and permissions, and monitor user activity across your organization. Maintain control over who has access to your asset management system.

## Key Features

- **User Invitations**: Invite team members via email
- **Role-Based Access**: Assign roles with specific permissions
- **Activity Tracking**: Monitor user actions and changes
- **Status Management**: Activate or deactivate user accounts
- **Permission Control**: Fine-grained control over feature access

## Quick Start

1. Navigate to the **Users** section from the main menu
2. Click **Invite User** to send an invitation
3. Enter the user's email and select their role
4. The user will receive an invitation email to join

## User Roles

Fluxera supports different user roles with varying levels of access:

### Owner
- Full access to all features
- Can manage billing and subscriptions
- Can delete the workspace
- Cannot be removed by other users

### Admin
- Full access to all features except billing
- Can invite and manage users
- Can create, edit, and delete assets and licenses
- Can view all reports and analytics

### Member
- Can view and create assets and licenses
- Can be assigned assets
- Can view reports
- Cannot manage users or settings

### Viewer
- Read-only access to assets and licenses
- Cannot create or modify data
- Can view reports and dashboards
- Ideal for stakeholders who need visibility

## User Statuses

Users can have the following statuses:

- **Active**: Full access to the system
- **Inactive**: Account disabled, cannot log in
- **Pending**: Invitation sent but not yet accepted
- **Suspended**: Temporarily restricted access
