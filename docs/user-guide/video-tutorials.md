## Video Tutorial Production Notes

**For Content Creators:**

When creating video tutorials, follow these guidelines:

### Technical Requirements
- Resolution: 1920x1080 (1080p minimum)
- Frame rate: 30fps or 60fps
- Audio: Clear narration with minimal background noise
- Format: MP4 (H.264 codec)
- Captions: Include closed captions for accessibility

### Content Guidelines
- Keep videos under 10 minutes when possible
- Start with a clear objective
- Use real-world examples
- Show keyboard shortcuts when applicable
- Include timestamps in description
- End with a summary and next steps

### Accessibility
- Provide closed captions
- Use clear, descriptive narration
- Ensure good color contrast
- Avoid flashing or rapid transitions
- Include transcript in description

### Branding
- Use consistent intro/outro
- Include Fluxera branding
- Display version number
- Add contact information for support
