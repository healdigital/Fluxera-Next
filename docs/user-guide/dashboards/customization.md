# Dashboard Customization

Learn how to personalize your Fluxera dashboard to display the information most relevant to your role and responsibilities.

## Overview

Fluxera allows you to customize your dashboard layout, choose which widgets to display, and configure how data is presented. This ensures you see the most important information at a glance.

## Configuring Widgets

### Adding Widgets

To add a widget to your dashboard:

1. Click **Customize Dashboard** button
2. Click **Add Widget**
3. Browse available widgets
4. Click on a widget to add it
5. Widget appears on dashboard
6. Click **Save Changes**

![Add Widget Dialog](../images/add-widget-dialog.png)

### Removing Widgets

To remove a widget:

1. Enter customization mode
2. Hover over the widget
3. Click the **X** icon in the corner
4. Confirm removal
5. Click **Save Changes**

### Rearranging Widgets

To change widget positions:

1. Enter customization mode
2. Click and drag widget by its header
3. Drop in desired position
4. Other widgets adjust automatically
5. Click **Save Changes**

### Resizing Widgets

To change widget size:

1. Enter customization mode
2. Hover over widget corner
3. Drag resize handle
4. Release at desired size
5. Click **Save Changes**

## Troubleshooting

**Q: Changes not saving?**
A: Ensure you click "Save Changes" before leaving customization mode.

**Q: Widget not displaying data?**
A: Check widget configuration, verify data source, refresh dashboard.

**Q: Can't rearrange widgets?**
A: Enter customization mode first, ensure you have edit permissions.

**Q: Lost my customization?**
A: Check if you saved changes, try loading from preset, contact support.

**Q: Dashboard looks different on mobile?**
A: Mobile uses separate configuration, customize mobile view separately.

**Q: Can I have multiple dashboard layouts?**
A: Yes, save different layouts as presets and switch between them.

**Q: How do I reset to default?**
A: Click "Reset to Default" in customization mode, confirm reset.

## Keyboard Shortcuts

Speed up customization with shortcuts:

- `Ctrl/Cmd + E`: Enter customization mode
- `Ctrl/Cmd + S`: Save changes
- `Ctrl/Cmd + Z`: Undo last change
- `Esc`: Exit customization mode
- `Ctrl/Cmd + R`: Reset to default
- `?`: Show all shortcuts
