# Dashboard Overview

Learn how to use the Fluxera dashboard to monitor key metrics, visualize data, and quickly access important information about your asset management system.

## Overview

The dashboard is your central hub for monitoring and managing your organization's assets, licenses, and users. It provides at-a-glance insights into key metrics, trends, and alerts that require attention.

## Dashboard Layout

The dashboard is organized into several key sections:

### 1. Metrics Summary

At the top of the dashboard, you'll find key performance indicators (KPIs):

**Total Assets**
- Complete count of all assets in inventory
- Includes all statuses (available, assigned, maintenance, retired)
- Click to view detailed asset list

**Assigned Assets**
- Number of assets currently assigned to users
- Percentage of total assets
- Indicates utilization rate

**Available Assets**
- Assets ready for assignment
- Buffer for new team members
- Click to view available assets

**Licenses Expiring Soon**
- Licenses expiring within 30 days
- Requires immediate attention
- Click to view expiring licenses

**Active Users**
- Number of users with active accounts
- Excludes deactivated and pending users
- Click to view user list

## Using Filters

### Date Range Filter

Control the time period for dashboard data:

1. Click the date range selector
2. Choose from presets:
   - Today
   - Last 7 days
   - Last 30 days
   - Last 90 days
   - This month
   - Last month
   - This year
   - Custom range
3. For custom range:
   - Select start date
   - Select end date
   - Click Apply

### Category Filter

Focus on specific asset or license categories:

1. Click the category filter
2. Select one or more categories
3. Dashboard updates to show only selected categories
4. Clear filter to show all

### Status Filter

View data for specific statuses:

1. Click the status filter
2. Select statuses to include:
   - Available
   - Assigned
   - In Maintenance
   - Retired
3. Dashboard updates accordingly
4. Clear to show all statuses

### Department Filter

Filter by organizational unit:

1. Click the department filter
2. Select department(s)
3. View department-specific data
4. Compare across departments

### Combining Filters

Use multiple filters together:
- Date range + Category
- Status + Department
- All filters combined
- Save filter combinations

## Exporting Data

### Export Dashboard

To export dashboard data:

1. Click the **Export** button
2. Choose format:
   - PDF (for reports)
   - Excel (for analysis)
   - CSV (for data processing)
   - PNG (for presentations)
3. Select what to include:
   - Metrics summary
   - Charts and graphs
   - Alerts list
   - Activity feed
4. Click **Download**

### Scheduled Exports

Set up automatic exports:

1. Click **Export** > **Schedule**
2. Configure:
   - Frequency (daily, weekly, monthly)
   - Format
   - Recipients
   - Filters to apply
3. Save schedule

### Export Uses

Use exported data for:
- Executive reports
- Board presentations
- Budget planning
- Compliance documentation
- Performance analysis

## Understanding Metrics

### Total Assets

**What it means:**
- Complete inventory count
- All assets regardless of status
- Baseline for other metrics

**What to watch:**
- Steady growth indicates expansion
- Sudden drops may indicate data issues
- Compare to budget and plans

### Assigned Assets

**What it means:**
- Assets in active use
- Utilization rate
- Resource allocation

**What to watch:**
- High percentage (>80%) indicates good utilization
- Low percentage may indicate over-purchasing
- Seasonal variations are normal

### Available Assets

**What it means:**
- Ready-to-deploy inventory
- Buffer for new hires
- Replacement pool

**What to watch:**
- Too many may indicate over-stocking
- Too few may cause delays
- Maintain 10-20% buffer

### Licenses Expiring Soon

**What it means:**
- Upcoming renewals
- Budget requirements
- Action items

**What to watch:**
- Plan renewals 60 days ahead
- Budget for renewal costs
- Evaluate continued need

### Active Users

**What it means:**
- Team size
- License requirements
- System usage

**What to watch:**
- Growth indicates expansion
- Ensure adequate licenses
- Monitor for inactive accounts

## Troubleshooting

**Q: Dashboard not loading?**
A: Check internet connection, refresh browser, clear cache.

**Q: Metrics seem incorrect?**
A: Verify filters are not applied, check date range, contact support if issue persists.

**Q: Charts not displaying?**
A: Enable JavaScript, update browser, check browser compatibility.

**Q: Can't export dashboard?**
A: Check permissions, ensure pop-ups are allowed, try different format.

**Q: Alerts not showing?**
A: Check notification settings, verify alert configuration, refresh dashboard.

**Q: Dashboard is slow?**
A: Reduce date range, apply filters, check internet speed, clear browser cache.
