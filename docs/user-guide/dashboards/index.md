# Dashboards & Analytics

The Dashboard provides a comprehensive overview of your asset management system with key metrics, visualizations, and quick actions. Get insights into your asset portfolio at a glance.

## Key Features

- **Metrics Overview**: View key statistics about assets, licenses, and users
- **Visual Charts**: Interactive charts showing trends and distributions
- **Quick Actions**: Fast access to common tasks
- **Alerts & Notifications**: Stay informed about important events
- **Customizable Widgets**: Arrange dashboard to suit your needs

## Quick Start

1. Navigate to the **Dashboard** from the main menu
2. View the metrics summary at the top
3. Explore the various widgets and charts
4. Click on any metric to drill down into details

## Understanding Metrics

### Total Assets
The complete count of all assets in your inventory, regardless of status.

### Assigned Assets
Number of assets currently assigned to users. High assignment rates indicate good utilization.

### Available Assets
Assets ready to be assigned. A healthy buffer ensures you can quickly equip new team members.

### Licenses Expiring Soon
Licenses expiring within the next 30 days. Take action to avoid service interruptions.

### Active Users
Number of users with active accounts. Monitor this to ensure proper license allocation.
